Resource: /api/devices

Topics:

  • Devices

The Device service facilitates creation and retrieval of measurement devices.

Actions

- (Object) index

Url:

  • [GET] /api/devices

List all of the device resources in the Safecast database

Required Arguments:

  • (String) where —

    Hash of search parameters a device must match to be returned: Valid fields in the hash are "manufacturer", "model", and "sensor"

  • (Integer) page —

    Results are paginated

  • (Integer) page_size —

    Number of devices to include in one page

- (Object) show

Url:

  • [GET] /api/devices/:id[.format]

Retrieve the device resource referenced by the provided id

Response Fields:

  • (String) manufacturer —

    The device's manufacturer

  • (String) model —

    The model number of the device, provided by the manufacturer

  • (String) sensor —

    The type or model of sensor element used in the device
