Resource: /api/measurements

Topics:

  • Measurements

The Measurement service facilitates posting and retrieving Safecast Measurements.

Actions

- (Object) index

Url:

  • [GET] /api/measurements

List all of the measurement resources in the Safecast database. There are a lot of measurements in the system (on the order of millions), so something this general may not actually be what you want.

Required Arguments:

  • (Integer) user_id —

    Indicate that results should only include measurements created by this user This can be provided manually, or by calling [GET] /api/users/:id/measurements

  • (Integer) page —

    Results are paginated automatically. Default is page 1.

  • (Integer) page_size —

    Number of devices to include in one page. Default is 10.

- (Object) show

Url:

  • [GET] /api/measurements/:id

Retrieve the measurement resource indicated by the provided id.

Required Arguments:

  • (Integer) user_id —

    Indicate that results should only include measurements created by this user This can be provided manually, or by calling [GET] /api/users/:id/measurements

  • (Integer) page —

    Results are paginated automatically. Default is page 1.

  • (Integer) page_size —

    Number of devices to include in one page. Default is 10.
